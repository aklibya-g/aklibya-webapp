# نظام التحويلات - شركة اليمامة المالية

## المتطلبات
- Python 3.8 أو أعلى
- اتصال بالإنترنت (لمرة واحدة فقط لتثبيت المكتبات)

## خطوات التثبيت على الجهاز الجديد

### 1. تثبيت Python
- حمّل Python من: https://www.python.org/downloads/
- **مهم:** حدد خيار "Add Python to PATH" أثناء التثبيت

### 2. نسخ المجلد
- انسخ مجلد النظام بالكامل إلى الجهاز الجديد

### 3. التثبيت والتشغيل
- اضغط مرتين على `install.bat` لتثبيت المتطلبات
- اضغط مرتين على `start.bat` لتشغيل النظام

### 4. فتح النظام
- افتح المتصفح وانتقل إلى: http://127.0.0.1:8000
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`

## الملفات
- `start.bat` - تشغيل النظام
- `install.bat` - تثبيت المتطلبات
- `requirements.txt` - قائمة المكتبات المطلوبة
- `db.sqlite3` - قاعدة البيانات (يتم إنشاؤها تلقائياً)

## ملاحظات
- النظام يعمل بدون إنترنت بعد التثبيت الأول
- قاعدة البيانات موجودة في المجلد (db.sqlite3)
- للحفاظ على البيانات، احتفظ بملف `db.sqlite3` في نفس المجلد
